# prompt-collection

本项目专注于收录常用 Prompts，为开发者和 AI 用户提供高质量、实用的提示词模板。

## 项目简介

prompt-collection 是一个开源的 Prompt 集合仓库，旨在收集和整理各类场景下的优质提示词，帮助用户更高效地与 AI 模型交互。

## 使用说明

1. 浏览本仓库中的 Prompt 文件
2. 根据需求复制相应的 Prompt 模板
3. 根据实际情况调整变量部分
4. 粘贴到 AI 对话中使用

## Prompt 分类

- 代码开发
- 文案写作
- 数据分析
- 学习辅助
- 创意生成
- 其他实用场景

## 贡献指南

欢迎提交 Pull Request 来分享您的高质量 Prompt！

1. Fork 本仓库
2. 创建您的分支 (`git checkout -b feature/prompt-name`)
3. 提交您的更改 (`git commit -am 'Add some prompt'`)
4. 推送到分支 (`git push origin feature/prompt-name`)
5. 创建 Pull Request
